# Apple Data Analysis Project

## Project Overview
This project includes data cleaning, transformation, modeling, merging, and dashboard development using Excel, Power Query, and Power BI.

## Datasets
- Sale_ID
- Sale_Date
- Quantity
- Claim_Date
- Repair_States
- Store_Name
- City
- Country
- Product_Name
- Product_Price
- Launch_Date
- Category_Name

## Tasks Completed
- Data Collection
- Exploratory Data Analysis (EDA)
- Data cleaning
- Data transformation
- Data modeling
- Merging tables
- Machine Learning (Time Series Forecasting)
- Dashboard creation
- Analysis insights

## Tools Used
- Excel
- Power Query
- Power BI
- Python
